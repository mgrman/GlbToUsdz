# GlbToUsdz
 
Tiny proof of concept of converting GLB files into USDZ files.

Only very limited feature set is supported:
- Basic Scene approximation (by doing decomposition of worldmatrices)
- Triangle meshes
  - Normals
- Diffuse Channel of Materials
  - Textures
